# はじめに

動画生成が発展して、すごそうな動画が増えてきました。  
でも「見ごたえのある動画」になるかというと、そうでもない。ここにきて **脚本（＝物語設計）がめちゃ重要** です。

そもそも何を伝えたい物語なのか？  
単純にそれっぽい、かわいい、かっこいいシーンを並べるだけになってないか？  
そういうのから卒業しましょう。これがこの記事の目的です。

「ホラーがつくりたいな」「地雷系の子で話つくりたいな」みたいなレベルから、もっと物語の構造を考えるために役に立つのが **「物語の型」** の知識です。

この辺がわかる図解を、具体例と一緒に解説した投稿を「後から見やすい」ようにまとめておきます。  
ひとつの型を知るだけでも、何十本と脚本アイデアは考えられるはず。  
同じ構造で「明治のハイカラ」「シリアスなスチームパンク」みたいに味付けを変えれば、すぐネタが増えます。

逆に、今つくっている物語が「なーんかいまいち」なら、こう聞けばいい。  
「この物語、Save the Catのジャンルだと何にあてはまる？」  
そのあと「そのジャンルだとして、足りないものは何？」  
不足してる要素が見えてきます。

例：
- 👹 **Monster in the House**：「出られないリアリティ」「モンスターの恐ろしさ」「罪の深さ」書けてる？
- 🏛️ **制度の中の物語**：狂った制度を支えるための「生贄」書けてる？
- 🔍 **ホワイダニット**（なぜそうなったのか）：動機を掘らずに密室トリックばっか考えてない？

もちろん脚本術は他にも色々ありますが、**Save the Catは基礎として持っておいて損はない**。  
物語ジャンルの解像度を上げて、無限に脚本を作ろう！

# Save the Catの法則 1. Monster in the House 👹🏠

**「出られない空間で」「対峙しないといけない怪物」がいる。  
でもそうなったのは、ある自分たちの過ち（罪）のせいだった…** という型。

例：
- シャイニング
- エイリアン
- ひぐらしのなく頃に（アニメ）
- 進撃の巨人（囲まれた閉鎖空間という意味で“味”がある）

ジャンルを決めると何がいいか？  
**絶対外したらいけないポイント** が見えてくる。

この場合の背骨：
- 🚪 **出られない理由**
- 👹 **怪物**
- ⚠️ **罪（過ち）**

まずこれを埋めると、必要キャラや展開が見えてくる。  
型を知ってると「ラブコメでMonster in the Houseできない？」「グルメなら？」みたいに発想が拡張できる。  
逆に「今の話がどの型か」分かれば、足りないピースがすぐ分かる。

# Save the Catの法則 2. 黄金の羊毛 🐑✨

定番中の定番。ギリシア神話イアソン（Jason）の型。  
分かりやすい例は **ロード・オブ・ザ・リング**。ワンピースもそう。  
大体、少年漫画・RPG的世界はこの感じ（努力友情勝利）。

だからこそ要所を外さないのが大事。

押さえるべき三点（1行で答えたい）：
- 🗺️ **なぜその旅に出るのか？**
- 👥 **どんな仲間がいるのか？**
- ⚔️ **どんな試練があるのか？**

注意：RPGっぽい＝自動的に黄金の羊毛、ではない。  
例：
- リゼロ：黄金の羊毛
- 転スラ：特に序盤は「問題をかかえた普通の人」
- オーバーロード（無双要素）：スーパーヒーロー
- フリーレン：黄金の羊毛が主軸だが、バディ・ラブ的要素も多い

型が違えば押さえるポイントも登場人物も変わる。  
分析するだけで時間が溶けるやつ！

# Save the Catの法則 3. 魔法のランプ 🪔✨

「願いを言え…」→ 願いが叶うが **代償** が増える型。  
好きな例：**シュタインズ・ゲート**（※未視聴の人にはネタバレになるかも、というノリ）

構造（シュタゲ例）：
1. 💫 **願望が手に入る瞬間がある**  
   主人公オカリンは「発明で世界を変えたい」「厨二の理想を現実にしたい」という“無自覚な願望”を持つ。  
   偶然「Dメールで過去を書き換える」能力＝ランプを入手。  
   → 願いが叶う瞬間（Call to Adventure）

2. ⚠️ **願いを叶えるほど代償が増える**  
   小さな成功が積み重なり世界線が歪む。  
   ついに大切な人（まゆり）が毎回死ぬという巨大な代償へ。  
   書き換えるほど状況が悪化する。  
   → 「望みが叶うほど代償が増大」こそ核心

3. 💡 **本当の願いに気づく（Break Into Three）**  
   最初は「世界を変えたい」「天才になりたい」みたいな未熟な外向き願望。  
   進むと「まゆりを救いたい」「紅莉栖を救いたい」「仲間を守りたい」へ到達。  
   → 最終工程

他にも：
- デスノート
- まどマギ
- ドラえもん（毎回これ）
- 異世界転生（大体この要素ある）
- 果てしない物語（ネバーエンディングストーリー）下巻はド直球

ここで大事：主人公の不満に感情移入しやすいほど入りやすい。  
異世界ものは最初から読者の不満を背負って旅立つのでカタルシスが速い。  
早めに代償が見えるとさらに入っていきやすい。  
味付け次第で無限に変化するので、外しちゃいけないポイントだけ押さえつつ遊び倒す！

# Save the Catの法則 4. 通過儀礼 🦋📚

主人公の **内面変化そのもの** が物語のエンジンになる型。  
例：
- あの日見た花の名前を僕達はまだ知らない（めんまの死、喪失と罪悪感の克服）
- 聲の形（贖罪と再生、自己否定からの人間関係再構築）

通過儀礼にありがちな要素：
- 💔 喪失・別れ・後悔・再生
- 🌱 思春期〜大人への移行
- ⚖️ 罪の意識・贖罪
- 💪 自分の弱さと向き合う
- 🌍 世界の見え方が変わる瞬間

逆に、外的世界の冒険が主軸（ワンピース、メイドインアビス）や、関係性の衝突→協力→信頼が中心（ぼっちざろっく、TIGER＆BUNNY）で「外側の出来事」が駆動するなら通過儀礼ではない。

このジャンルの最重要：  
**「主人公は現実から目を背けている」** スタートをしっかり描くこと。  
拒否してるから変化が壁になる。拒否状態が燃料。

必要なのは「変化を迫る圧力」。  
例：イヴの時間  
「アンドロイドと人間的な関係はキモイ」世界で、区別がつかないカフェの事件が主人公の価値観を更新し、親友とのギャップが拡大していく。現代っぽい通過儀礼。

クライマックスは **受容**。  
🚫 拒否 → 💥 圧力 → ✅ 受容 の3点セット。

# Save the Catの法則 5. 問題を抱えた普通の人 😰💪

「なんで俺がこんな目に…」型。例：**ダイ・ハード**。  
理不尽な環境に放り込まれた普通の人が、創意工夫で生還する。  
普通の人だから感情移入しやすいし、「逃げ場がない」緊張感で視線を支配できる。

Monster in the Houseと違い、このトラブルは **罪への罰ゲームではない**。理不尽であるほどいい。  
通過儀礼と違い、勝ち筋は **心理的成長ではなく工夫**。

ダイ・ハード（マクレーン）：
- 裸足
- 装備貧弱
- 仲間なし
- 思いつき・工夫・逆手・アドリブ・環境利用で切り抜ける

観客の快感：  
「俺ならこうする…あ、やった！」  
→ 脳内シミュレーションと行動が一致する中毒性。

アニメで典型は少なめ（日本は心理的成長が多い）が、**カイジ**がド真ん中。

カイジ：
- 💸 借金あり
- 🏠 無職
- 📉 社会的弱者
- ⚔️ 戦闘力ゼロ
- 😓 メンタル強くない（でも覚醒する）

解決が内面でなく物理的戦略：
- 📖 ルールを読む
- 🧠 相手心理を読む
- ⚡ 状況のギリギリを突く
- 💡 閃き・工夫

この構造で大事：
- 👤 「いかに普通のやつか」を徹底演出
- 💥 とんでもない問題
- 🚪 逃げれない状況を明確に

例：エスポワール号（逃げれない）→地下（もっと逃げれない）…の連鎖が典型。

# Save the Catの法則 6. バディラブ

正反対の2人が反発しつつ補い合い、別離と再会や葛藤を経て高みに到達する。  
恋愛である必要はなく、**2人の関係性の変化が推進力** になる。

典型例：
- ハイキュー
- TIGER&BUNNY
- サイコパス
- 響け！ユーフォニアム
- かぐや様は告らせたい

例：サイコパス  
狡噛慎也と常守朱音は刑事の相棒として典型だが価値観が深く食い違う。  
「法（シビュラ）は正しいのか？」が核で、事件が揺さぶる。  
槙島聖護と狡噛慎也はシビュラを信じない点では同側面もある。  
常守朱音が対比（名前も深夜-茜-正午…という話）。  
暴走する狡噛と未熟な朱音が補い合い、別離と再会を経て変容・再統合し、シビュラと対峙する。

例：かぐや様  
刑事でも共同戦線でもないがジャンルは同じ。  
正反対の2人が、補い合い、すれ違いと葛藤の末に高みに到達する（告白など）。

正反対でもバディラブではない例：
- スラムダンク：流川と花道（クライマックスはバディっぽいが全体では関係性が大きく変化しない）
- シンジとアスカ（途中までバディっぽいが肯定的再構築ではなく破滅へ向かうので違う）

面白い例：フリーレン  
大筋は黄金の羊毛だが、各所にバディラブが散りばめられてテンポと感情移入が強い。  
例：フリーレン×フェルン、フリーレン×ヒンメル（死後に成立する変化球）、師弟関係など。

# Save the Catの法則 7. スーパーヒーロー 🦸

「力」を持つがゆえの葛藤と責任を描く型。  
例：スーパーマン、バットマン、スパイダーマン（アメコミ）。  
日本のアニメ：ヒロアカ、TIGER&BUNNY、ワンパンマン、チェンソーマン、デビルマン…  
（日本の方が力の代償＝内面的葛藤に重きを置きがち）

ヒーローは「自分がこうありたい」の投影。  
スーパーヒーローは理想像の濃縮：
- 💪 強さ
- 💚 優しさ
- ⚖️ 道徳性
- 🎯 決断力
- 📋 責任

日常では発揮しきれないが、ヒーローを見ると代わりに体験できる。  
だから世相も映る。社会不安が強いとヒーローが生まれやすい、という話もある。

スーパーヒーローに必要な要素：
- ⚡ 力（gift）
- 🎭 正体（identity）
- ⚖️ 責任と対立（responsibility and nemesis）

力に集中しがちだが、同じくらい **日常生活** が大事。  
なぜならIdentity（正体の揺らぎ）は日常でしか描けない。

Superheroの最大テーマ：  
**「自分は誰なのか？」というアイデンティティ葛藤**。

日常で起こる矛盾の例：
- 🏠 家に帰った瞬間
- 🏫 学校や仕事で普通に過ごすとき
- 🎭 嘘をつく／隠すとき
- 💔 力のせいで約束を破るとき

例：
- スパイダーマン：街を救うとデートを台無しにする
- バットマン：正義のためにブルースの人生が壊れる
- チェンソーマン：人間の欲望があるのに悪魔として扱われる

そして宿敵ヴィランは、同じ「力」を持ちながら違う選択をした **主人公の鏡像**。  
決着は善悪だけでなく、力に対する倫理観の決着であるべき。

価値観の決着の例（ヒーローっぽくないが本質は近い）：
- るろうに剣心：剣心 vs 志々雄（守るため vs 支配するため）
- 鋼の錬金術師：エド vs 父（個の命 vs 数／目的）

# Save the Catの法則 8. ホワイダニット（なぜやったのか？）🔍❓

ミステリー＝全部ホワイダニット、ではない。  
この3つは全部ミステリーだが、ホワイは1つだけ：
- 名探偵コナン（Who：誰がやったか＝犯人当て）
- Monster（Why：なぜやったか＝動機）
- Death Note（How：どう暴くか＝倒叙・攻防）

Death Noteは夜神月が犯人だと最初から分かるのでWhoではない。  
Whyも新世界の神になる…で主眼が違い、How（どう追い詰める・どう破綻する）側。

Monsterがホワイの典型。  
「ヨハン！なんでお前そんなこと…！！！」の連続で、最後にゾッとする吐露や「“どっちだったの？”」という問いが事件全体を動かしていたと分かる。

昭和落語心中も殺人事件があり「なぜそうせざるを得なかったのか」を追う。  
中心に「死神」の落語が鎮座する、という話。

Who / How / Why を最初に決めないとブレる。  
それぞれ押さえるべきポイントを整理。

## ① Whodunit（誰がやったか）
- 物語の推進力：犯人当ての「情報パズル」
- 観客は「容疑者リスト」「手がかり」を見て推理する

押さえるべきポイント：
- 容疑者の人数・立場の差別化（混ざると終わり）
- フェアな手がかり配置（後出し禁止）
- 誤誘導（レッドヘリング）
- 犯人が判明した瞬間がピーク
- 内面よりロジックの満足度が重視
- 「パズル完成感」で勝負

## ② Howdunit（どうやったか／倒叙ミステリ）
- 推進力：犯行トリックや捜査の過程
- 観客は犯人を知った上で「どう暴く？」「どこでボロが出る？」を楽しむ

押さえるべきポイント：
- 👤 犯人視点が主体
- 🎨 犯行の美しさと綻び
- 🔍 主人公（刑事）が追い詰める段階設計
- 💥 クライマックスは論理崩壊の瞬間
- 🧠 感情より知的興奮が重視
- コロンボ式＝「美しい負け方」を描くのが肝

## ③ Whydunit（なぜやったか）
- 推進力：動機のドラマ
- 観客は「犯人の心の闇」「価値観の対立」に惹かれる

押さえるべきポイント：
- 💎 犯人の信念を最初から核に置く
- 🪞 主人公と犯人が鏡の関係
- 💥 真相解明＝感情爆発になる構造
- 🤔 ラストで観客の価値観を問い直す
- 🔄 理由が分かった時に全シーンが再解釈される
- トリックが弱くても成立しやすいのが強み

ホワイダニットの典型っぽいやつ（どこがそうか推理してみて、という投げ）：
- 💜 ヴァイオレット・エヴァーガーデン
- 🐺 狼と香辛料
- 🧙 葬送のフリーレン

# Save the Catの法則 9. 愚者の勝利 🤡👑

一番有名で純度高い例：フォレスト・ガンプ。

愚者の勝利の構造：
- 🤡 主人公は社会不適合の愚者
- 🏛️ 敵は「社会制度／秩序／常識」
- 💫 勝因は「愚者性（愚直・無垢・反合理）」
- 🌍 結果として制度や価値観が変容する
- ⚠️ 主人公は勝つが、報われないこともある

重要なのは「愚者ゆえ勝つ」「世界のほうが変わる」カタルシス。

好きな映画例：I am Sam  
知的障害をもつシングルファザーが娘と引き離され、裁判で争う。  
サムは変わらないが折れない。周囲（弁護士リタなど）が感化されて変わっていく。

アニメだと少なめ（成長が主流）だが、典型は銀魂。  
社会性がない銀時が、天人支配体制・腐敗幕府・組織論理に喧嘩を売る。  
政治的／風刺的になりやすい型をコメディに落としたので見せやすい。  
ワンピースも純度は低いが特徴を持つ。

# Save the Catの法則 10. 制度の中の物語 🏛️⚖️

分かりやすい例：**半沢直樹**。  
この型に必要な「制度」「選択」「生贄」が揃っている。

## 1) 🏛️ 制度（The Group）＝東京中央銀行
閉鎖的集団が独自の狂ったルールで運営されている必要がある。  
半沢のルール例：  
「部下の手柄は上司のもの、上司の失敗は部下の責任」  
銀行という組織の利益のためなら個人の人生や取引先の運命を平気で踏みにじる。  
外から見れば異常だが、中では絶対的掟。

## 2) ⚡ 選択（The Choice）＝倍返し
主人公は制度に対し「順応するか」「脱出するか」を迫られる。  
面白いのは「制度の中に留まりながら腐敗と戦う（自殺行為）」を選ぶパターン。

半沢は銀行を辞めない。  
出向（追放）という死刑宣告を突きつけられながらも内部から正義を貫く。  
「倍返し」は組織の掟へのカウンターパンチ。

## 3) ⚠️ 生贄（The Sacrifice）＝敗れ去るライバルや同僚たち
制度維持のために誰かが犠牲になる。  
敵役でさえ組織という巨大な怪物に飲み込まれる／守るために頭を下げる。  
視聴者は「制度」という怪物が個人によって一時的にでもひれ伏す姿にカタルシスを感じる。

アニメ例：銀河英雄伝説  
「銀行」が「宇宙軍」になったと考えると近い。

制度（組織）：
- 銀河帝国（独裁制）
- 自由惑星同盟（腐敗した民主主義）

主人公：
- ラインハルト（帝国側）：腐った貴族社会を憎み、内部から実力で登り詰める野心家（倍返しの塊）
- ヤン・ウェンリー（同盟側）：腐敗を嫌い辞めたいのに、才能ゆえ守らざるを得ない中間管理職的悲哀

見所は艦隊戦だけでなく、会議室の政治劇や人事抗争。  
「無能な上官の尻拭い」に共感しやすい。

銀河英雄伝説における「生贄」例：
- 帝国側：ジークフリード・キルヒアイス  
  ラインハルトの親友で良心でブレーキ役。  
  制度の論理（政治的陰謀）を受け入れる過程で凶刃に倒れ、ラインハルトが孤高の皇帝へ変貌するための代償となる。

- 同盟側：ジェシカ・エドワーズ（およびラップ）  
  市民としての正義の象徴。  
  制度の軋轢の中で命を奪われ、ヤンが守る「民主主義」という制度の脆さと矛盾を突きつける。  
  それでも制度を守るため戦い続けねばならないパラドックスを生む。

制度の中の物語で大事なのは：  
「狂った制度を守るために何が犠牲になっているのか」  
「その中で主人公は何を選択するのか」

進撃の巨人、PSYCHO-PASS、SHIROBAKO なども毛色は違うが近い。  
職場やコミュニティの狂った制度も、このフレームに落とすと脚本のネタになるかも。
