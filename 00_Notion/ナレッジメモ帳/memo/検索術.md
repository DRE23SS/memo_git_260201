      - LOBSTERR キュレーターベース

    - 良い検索方法を検索する＝メタ調査

      |               |                                |
      | ------------- | ------------------------------ |
      | **–**         | 特定のキーワードを除外して検索 |
      | **”TEXT“**    | 完全一致のキーワードで検索     |
      | **OR**        | いずれかのキーワードを含む検索 |
      | **AND**       | どちらのキーワードも含める検索 |
      | **site:**     | 特定のサイト内を検索           |
      | **related:**  | 関連するサイトを検索           |
      | **intitle:**  | キーワードを含むタイトルを検索 |
      | **filetype:** | ファイル形式で検索             |
      | **cache:**    | 最新のキャッシュページを検索   |
