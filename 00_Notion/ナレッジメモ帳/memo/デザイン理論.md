- デザインの要素

  - ビジュアル

  - タイポグラフィ

    - 可読性高い＝明朝体・ゴシック体

      - 本文＝細い書体を使用する

    - 装飾性高い＝その他の書体

  - レイアウト

  - カラー
    - デザイン＝ 4 要素をコントロールすること

  ***

- 「わかりやすい」を伝える

  - 見やすい＆読みやすい

    - デザインフォント

      - 目的を持って使用する

      - 本文に使用しない

  - メインビジュアルのみ

  - ベーシックフォント

    - 明朝体　和風・伝統・高級

    - ゴシック体　洋風・モダン・カジュアル

    ![[DR-TypeKeyword.pdf]]

  - レイアウト　上から下

  - カラー　 1 色のみ
    - 無彩色　黒・白・グレー
      - 明度の差が大きいほど目立つ

- 文字の大きさ

  |              |         |                |
  | ------------ | ------- | -------------- |
  | 文字の大きさ | Q（級） | Pt（ポイント） |
  | タイトル     | 適宜    | 適宜           |
  | リード文     | 12~15   | 8.5~11         |
  | 小見出し     | 14~16   | 10~12          |
  | 本文         | 11~15   | 8~10.5         |
  | キャプション | 8~10    | 6~7            |

- 行長

  1 行の長さ＝ 15 ～ 40

  段落分けする

- 行送り

  フォントサイズの「1.5 倍」～「2 倍」

- マージンが狭いと力強い印象になる

  - 行間も文字間も同様

- 余白の効果

  - 視線を集中させる

  - 圧迫感を軽減する

  - 清潔感・高級感を生み出す

- ==3B の法則==

  - 美人・赤ちゃん・動物（3B）

    - 視線を惹きつけやすい

    - 好感を持たれやすい

- デザインの 4 大原則

  - 整列

    - 並び替え

    - タテ・ヨコを揃える

    - 全てを揃える必要はない

  - 近接

    - グループ化

    - 関係性の強いモノを近づける

  - 反復

    - オブジェクトの間隔が同じ

    - 各要素も同じ

    - 色による反復
      - 各要素の色が別々

  - 対比

    - メリハリ・コントラスト

    - 大きさ・太さ・色

    - 大きさの違い＝ジャンプ率

- 安心感のあるデザイン

  - カラー

    - ピンク　薄め

    - 水色　薄め

- 高級感のあるデザイン

  - 余白を多くとる（大前提）

  - 色・金・銀・黒

  - 小さい文字で文字間を広く

  - ベーシックフォントを使用する

  - おすすめ　欧文書体

    - HELVETICA 　 condensed

    - DIN Narrow

    - TRAJAN 大文字のみ

    - GARAMOND

      - セリフ体

      - サンセリフ体

  - 視線
    - 人間は目を最優先で認識する（モデルは正面を向ける）
