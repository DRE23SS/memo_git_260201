##  服装・ファッション表現（MECE：ジャンル別）
> 重要になった点（原文主張）  
> - 素材感（布・レザー・金属）の描き分け  
> - レイヤード（重ね着）構造  
> - アクセサリーや小物の自然な馴染み  

### A. 男女兼用・定番（ベース装備系：検証/量産向き）
- ねらい：世界観を邪魔しにくい。シンプルほど表情・ポーズ・世界観が立つ。
- 特徴：キャラ性を壊しにくい／流用しやすい／色指定で印象を変えやすい

```text
Suit set / スーツセットアップ / ビジネスやフォーマルな場面に合う標準的なスーツスタイル
Tuxedo / タキシード / 結婚式やパーティーで用いられる格式高い礼服
Butler uniform / 執事服 / ジャケットにベストと蝶ネクタイを合わせた執事風衣装
Nurse uniform / 看護師服 / 医療従事者を象徴するシンプルな白衣やナース服
School uniform / 学生服 / ブレザーやセーラー服など定番の制服スタイル
Gakuran / 学ラン / 日本の男子学生に多い立ち襟の黒い制服
Police uniform / 警察官の服装 / 帽子・階級章・ベルト付きの警察官制服
Spacesuit / 宇宙服 / 宇宙飛行士が着用する近未来的で機能的なスーツ
Taisho modern / 大正モダン / 和洋折衷のモダン、着物＋帽子/袴＋洋装小物
Meiji fashion / 明治ファッション / 和装と洋装が混在、ドレス/スーツ＋和小物
Meiji military uniform / 明治時代の軍服 / 詰襟や肩章つきの洋風軍服
````

### B. SF・ファンタジー（男女兼用：服装が設定を語る）

- ねらい：メカ要素・発光素材・装甲×布の融合が安定しやすい、という主張。
    
- 検証ポイント（原文）
    
    - 「light-emitting」「mechanical」「ornament」が効きやすい
        
    - 全身指定より「upper / lower」で分けると破綻しにくい
        

```text
Cyberpunk fashion / サイバーパンク / ネオンカラーやメカニカルなアクセの近未来衣装
Steampunk fashion / スチームパンク / 機械×ヴィクトリア朝様式の衣装
Plague doctor / ペストマスク / 錬金術師や闇医者を思わせる仮面スタイル
Medieval armor / 中世の鎧 / 鉄のプレートやチェーンメイルを備えた騎士風防具
Victorian style / ビクトリア朝様式 / 19世紀英語圏中心の様式
Renaissance / ルネサンス / 15世紀〜17世紀初頭の様式
```
