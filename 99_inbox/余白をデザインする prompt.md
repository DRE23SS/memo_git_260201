# White Space Design: 6 Principles for Professional Layout

| 制限対象 (Target)            | 指示の内容 (Instruction)    | 得られる洗練さ (Effect)  |
| :----------------------- | :--------------------- | :---------------- |
| **Hierarchy Control**    | 太字を封印し、サイズと位置のみで階層を作る  | 装飾に頼らない構造的な美しさ    |
| **Vertical Axis**        | 起点を極少数の垂直軸に限定し、吸着させる   | 迷いのない視線誘導と強い清潔感   |
| **Modular Scale**        | 数値を特定の比率（黄金比等）のみに縛る    | 画面全体の数学的な調和とリズム   |
| **Negative Shape**       | 余白を先に「主役の図形」として配置する    | 大胆で贅沢な空間のポテンシャル   |
| **Baseline Grid**        | すべての行を一定の垂直リズムに完全吸着させる | 建築的な秩序とアカデミックな信頼感 |
| **Borderless Proximity** | 枠線を廃し、要素間の「距離」のみで分類する  | ノイズが極小化されたモダンな透明感 |
